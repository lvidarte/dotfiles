# vim

#### Split

    Ctrl-o    split horizontal
    Ctrl-e    split vertical

#### Change focus

    Ctrl-h    focus left
    Ctrl-j    focus down
    Ctrl-k    focus up
    Ctrl-l    focus right

#### Resize

    Shift-h    resize left
    Shift-j    resize down
    Shift-k    resize up
    Shift-l    resize right

#### Misc

    Ctrl-i    join lines

#### Fuzzy Finder

    sj    search buffer
    sk    search file
